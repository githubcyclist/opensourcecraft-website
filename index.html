Get it

MediaFire - Latest Compiled (Recommended)
Please read the instructions.

Running this is very easy: simply extract the archive, and double-click on the DoubleClickToStart.jar file.
If that doesn't work, open a terminal or command prompt.
On Windows, open the extracted archive in Explorer, right click while holding the Shift key, and select Open command window here.
On Linux, right click in the extracted archive and select Open in terminal.

After that's done, type in this command: java -jar DoubleClickToStart.jar
